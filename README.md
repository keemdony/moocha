# 🌟 Project Name —  공차 페이지 리브랜딩 리뉴얼 'Moocha'

> *MOOCHA (무차) : ‘무(無)’ + ‘차’ 잡념 없이 마시는 차 / 미니멀, 젠한 감성

---

## 🧩 프로젝트 소개
고객의 시각적 만족도를 극대화하기 위해 영상 콘텐츠, 감각적인 레이아웃 구성,
직관적 인터페이스 디자인을 적극 활용했습니다.
동적(動的) 요소와 사용자 중심(UI/UX) 설계를 통해 브랜드의 감성과 메시지를 효과적으로 전달하며, 
방문자가 브랜드 경험을 직접 체감할 수 있는 몰입형 환경(immersive environment) 을 구현했습니다.

---

## 🚀 주요 기능
- 🚨 이벤트 / 공지사항 
- 📝 앱다운로드 / 매장 정보 / 가맹문의
- 💬 고객센터 게시물 작성
- 📱 반응형 UI (모바일/데스크탑 지원)

---

## 💡 페이지 설명
- 매장을 이용하려는 고객을 대상으로 브랜드 소개 
- 주 원재료 설명, 메뉴 종류, 메뉴 설명 및 영양정보   
- 매장 위치, 영업시간, 편의시설, App 다운로드 정보
- 음료 구매에 실질적인 도움을 주는 서비스를 제공합니다.
